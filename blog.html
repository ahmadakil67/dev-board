Question 1: What are the different ways to select an element in the DOM?

We can select elements using getElementById for unique IDs, getElementsByClassName or querySelectorAll for classes, and getElementsByTagName or querySelectorAll for tag names.


Question 2: What is the difference between innerHTML, innerText, and textContent?

innerHTML parses HTML, innerText gets visible text, and textContent gets all text content, including hidden.


Question 3: What is event delegation in the DOM?

Event delegation uses a single parent element listener to handle events for all child elements, improving efficiency.


Question 4: What is event bubbling in the DOM?

Event bubbling is when an event on a child element travels up the DOM tree to its parents.


Question 5: How do you create, add, and remove elements using JavaScript?

Create elements with createElement, add them with appendChild or insertBefore, and remove them with removeChild or remove.
